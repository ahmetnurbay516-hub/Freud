# 🧠 FREUD — Solana Psychological Profiler

> *Your wallet is your unconscious mind.*

FREUD reads your Solana on-chain history and exposes the FOMO, panic, revenge, and emotional instability behind every trade.

🔗 **Live:** [freud-phi.vercel.app](https://freud-phi.vercel.app)

-----

## ✨ Features

- **🔍 Wallet Scanner** — Real on-chain data via Helius RPC
- **🧠 Psychological Profiling** — 9 behavioral scores calculated from real metrics
- **🎭 10 Archetypes** — FOMO Chaser, Rug Magnet, Diamond Hand and more
- **🔥 Roast Me** — AI-generated roast based on your actual trading behavior
- **🛋️ Therapy Session** — Dr. FREUD gives you a personalized therapy plan
- **⚔️ Wallet Battle** — Compare two wallets, FREUD declares a winner
- **🏆 Leaderboard** — Hall of Shame & Fame
- **🖼️ Save Card** — Download your profile as a shareable image

-----

## 🔬 How It Works

```
1. FETCH   → Helius RPC pulls SOL balance, token accounts & transaction history
2. ANALYZE → Mathematical scoring engine calculates FOMO, panic, meme addiction
3. PROFILE → Archetype mapping + behavioral commentary from real on-chain data
```

## 🚀 Getting Started

```bash
git clone https://github.com/ahmetnurbay516-hub/Freud
cd Freud
npm install
npm run dev
```

Add your Helius RPC key to environment variables:

```
VITE_HELIUS_RPC_URL=https://mainnet.helius-rpc.com/?api-key=YOUR_KEY
```
